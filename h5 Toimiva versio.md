## Tiivistelmä

- Git on sovellus jonka voi ladata linux tietokoneelle joka yhdistää github hakemistossa kloonaamalla sen omalle tietokoneelle.
- Sillä voi luoda, poistaa, lukea ja muttaa kätevästi terminaalista.
- Jokaista muutoksesta kohden git tulostaa muutoslogin, muutoksia voi tallentaa tai poistaa kokonaan, tätä kutsutaan versiohallinnaksi.
  
- git add . on komento joka valmistelee kaikki muutokset nykyisessä hakemistossa ja alaspäin.
- git commit ottaa edelliset valmistellut muutokset ja säästää ne reposition historiaan.
- git pull vetää hakemistostasi kaikki ajantasaiset tiedostot, tee tämä aina enne pushia.
- git push vie kaikki commitin muutokset github hakemistoon.
